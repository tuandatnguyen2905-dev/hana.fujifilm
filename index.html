Đồng bộ qua GitHub Gist
Lưu dữ liệu vào GitHub Gist — miễn phí, đồng bộ tức thì giữa mọi thiết bị. Chỉ cần tài khoản GitHub.
1
Vào github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Nhấn Generate new token → Tích vào quyền gist → Generate → Sao chép token
2
Dán token vào ô bên dưới. Nếu bạn chưa có Gist, để trống ô Gist ID — hệ thống sẽ tự tạo. Nếu đã có Gist từ thiết bị khác, dán Gist ID vào để dùng chung.
💡 Dùng chung trên nhiều máy: Sau khi kết nối trên máy đầu tiên, sao chép Gist ID (nhấn 📋) → Dán vào máy thứ hai cùng với token → Nhấn Kết nối
